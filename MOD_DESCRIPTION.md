# Adaptive Nemesis / 自适应宿敌

> **"Your power feeds their evolution"**  
> *你的力量滋养着它们的进化*

---

## 简介

**Adaptive Nemesis（自适应宿敌）** 是一个专为 Minecraft 整合包设计的 **NeoForge 1.21.1** 动态难度平衡模组。

它解决了整合包中常见的 **"前期刮痧、后期秒天秒地"** 的难度失衡问题，通过智能评估玩家强度并动态调整敌人属性，让战斗始终保持 **"有点难，但不多"** 的紧张刺激感。

---

## 核心机制

### 1. 玩家强度评估系统

模组会持续监测每位玩家的综合战斗力，基于多维度数据进行评估：

| 评估维度 | 说明 |
|---------|------|
| 防御能力 | 护甲值、血量上限、护甲韧性 |
| 输出能力 | 攻击力、攻击速度、武器附魔等级 |
| 神话词条 | [Apotheosis](https://www.curseforge.com/minecraft/mc-mods/apotheosis) 装备品质与等级 |
| 铁魔法 | [Iron's Spells 'n Spellbooks](https://www.curseforge.com/minecraft/mc-mods/irons-spells-n-spellbooks) 法力值、法术强度 |
| 史诗战斗 | [Epic Fight](https://www.curseforge.com/minecraft/mc-mods/epic-fight-mod) 耐力值、战斗风格 |

> 所有权重均可通过配置文件自定义，支持 KubeJS 脚本扩展评估逻辑！

### 2. 敌人动态强化

当敌对生物生成时，模组会根据 **附近玩家的平均强度** 为其提供自适应属性加成：

- 血量提升（最高 500%）
- 伤害提升（最高 500%）
- 护甲提升（最高 300%）
- 攻击速度提升（防止被无限硬直）
- 法术强度/抗性（铁魔法兼容）
- 受击抗性/击倒抗性（史诗战斗兼容）
- 耐力值（史诗战斗兼容）

**属性随机分布**：每次生成的敌人属性会在基础值上下浮动（70%~130%），让战斗充满变数！

### 3. 真实伤害转化（铁乌龟终结者）

针对高护甲玩家，模组会将部分伤害强制转化为 **无视护甲的真实伤害**：

| 护甲等级 | 真实伤害比例 |
|---------|------------|
| 标准护甲（≤20） | 5% |
| 强化护甲（20~50） | 15% |
| 高护甲（50~100） | 25% |
| 铁乌龟（>100） | 35% |

> 再也不怕穿上神装就无敌了！敌人会找到你的弱点 🔥

### 4. 智能浮动系统

根据玩家的实时表现动态调整难度：

- **连续击杀** → 浮动倍数 +10%（敌人越来越强）
- **频繁死亡** → 浮动倍数 -15%（给你喘息机会）
- **长时间未战斗** → 浮动倍数重置为基准

### 5. 宿敌记忆系统

每个玩家都有独立的 **宿敌档案**，敌人会记住你的战斗方式并针对性进化：

- **击杀偏好**：你常用的击杀手段 → 敌人获得对应抗性
- **死亡记录**：你死亡时的伤害来源 → 敌人学会对应攻击方式
- **装备历史**：你曾使用过的装备组合 → 敌人针对性进化反制策略
- **行为模式**：近战/远程/法术偏好 → 敌人调整 AI 行为

### 6. 新手保护机制

为低强度玩家提供贴心保护：

- 玩家综合强度 < 阈值 → 怪物属性 -30%
- 首次死亡 → 保护时间 +10 分钟
- 连续死亡 3 次 → 强制开启保护直至击杀任一敌人

### 7. Boss 特殊机制

防止玩家秒杀 Boss，延长史诗战斗体验：

- **伤害上限**：单次伤害存在上限（默认 100），Boss 血量越低上限越高
- **属性增幅**：Boss 生命 ×5、伤害 ×3
- **战斗追踪**：记录战斗时长与累计伤害

Adaptive Nemesis 原生支持以下热门模组：

| 模组 | 兼容内容 |

## 游戏内命令

所有命令以 `/an` 为前缀，需要 OP 权限（等级 2）：

| 命令 | 说明 |
|-----|------|
| `/an status` | 查看模组当前运行状态 |
| `/an strength [玩家]` | 查看指定玩家的强度评估数据 |
| `/an difficulty` | 查看/调整难度设置 |
| `/an protection [玩家]` | 查看/管理新手保护状态 |
| `/an memory [玩家]` | 查看宿敌记忆档案 |
| `/an scan [范围]` | 扫描周围敌人的加成数据 |
| `/an nemesis [类型]` | 召唤宿敌 |
| `/an test [模块]` | 测试模组各模块功能 |
| `/an reload` | 重新加载配置文件 |
| `/an help` | 显示帮助信息 |

## 配置文件

模组的所有机制都可通过配置文件精细调整，主要配置分类：

- **基础难度**：整体难度倍率
- **真实伤害**：护甲阈值与转化比例
- **Boss 机制**：伤害上限、属性增幅
- **智能浮动**：浮动范围、击杀/死亡调整量
- **新手保护**：阈值、持续时间、减免比例
- **加成上限**：各项属性的最大倍率
- **强度权重**：各维度评估权重
- **多人联机**：区域同步范围
- **随机分布**：属性波动范围

配置文件路径：`config/adaptive_nemesis.toml`

## 设计哲学

> *"你的力量滋养着它们的进化"*

Adaptive Nemesis 的核心理念是 **动态平衡**：

1. **不削弱玩家** —— 玩家依然可以享受变强的快感
2. **强化敌人匹配** —— 让敌人始终与玩家实力相匹配
3. **学习并反击** —— 敌人会学习玩家的战斗风格并针对性进化
4. **保护新手** —— 给新玩家足够的成长空间
5. **挑战高手** —— 让高手玩家始终有挑战性

无论是初入整合包的新手，还是一身神装的老玩家，Adaptive Nemesis 都能为你提供恰到好处的战斗体验！⚔️
